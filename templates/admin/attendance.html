Bulk Upload Attendance

You can bulk record attendance logs by uploading a CSV spreadsheet. Perfect for loading weekly or monthly batch rosters.

Expected CSV Header Layout:

Ensure your CSV file has the exact headers below (the columns can be in any order):

username, date, status, remarks
  • username: Teacher's system login ID OR Student's Unique ID (e.g. AHM-XXXXXX)
  • date: Calendar date in YYYY-MM-DD format
  • status: Present, Absent, Late, Half Day, or On Leave
  • remarks: Optional textual description or leave note.
